## Matlab wrapper for MIRE

### MIRE Program :

If you want to use the mire program :
- Go to the current directory with Matlab
- Within Matlab, type :

>compileMex

You can now use the mire function within matlab

Type :
  > help mire  

in Matlab to see how it works.
