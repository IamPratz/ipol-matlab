## Matlab wrapper for SSDD

### SSDD Program :

If you want to use the SSDD program :
- Go to the current directory with Matlab
- Within Matlab, type :

>compileMex

You can now use the SSDD function within matlab

Type :
  > help ssdd  

in Matlab to see how it works.
